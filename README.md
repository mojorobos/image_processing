# Processing
> Processing is an edge detection tool part of a Rembrandt-style robotic arm solution developed by Mojo Robos

### Installing
  - `cd /path/to/image_processing/`
  - `make`

### Usage
  - `./processing "path/to/imageIn.jpg" "path/to/imageOut.jpg"`
